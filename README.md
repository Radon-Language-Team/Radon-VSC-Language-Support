# Radon VSCode Language Support
The official Language Support made for the Radon language. 
## Features
- Comments
    - Singleline comments: `!!`
    - Block comments: `!!*` `*!!`
- Variables
    - `$int`
    - `$string`
    - `$char` 
- Brackets
    - `()`
- Keywords
    - `if/while/for/quit/var`
    - `log`
